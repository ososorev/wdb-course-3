1. Форма регистрации строится с помощью Javascript (в html у вас пустой body). Запрос на регистрацию уходит с помощью AJAX. Код должен быть оформлен в виде замыканий (https://learn.javascript.ru/closure). Нужен автоматический переход на страницу входа (используем Javascript window.location.href = ...)

2. Форма входа делается с помощью HTML-форм. Пересылка данных имеено с помощью форм (без AJAX). Для запоминания текущего пользователя используем сессии PHP (https://habr.com/ru/post/437972/). После успешного входа отправляем пользователя на страницу записной книжки (используем PHP header("Location: ..."))

3. Страница записной книжки. Основа делается с помощью html-разметки. Поведение такое:

- при клике на заметку в списке заметок уходит AJAX запрос на сервер. После получения ответа данные о заметке рисуются с помощью Javascript в правой части экрана;

- при клике на карандашик у заметки уходит AJAX запрос на сервер. После получения ответа рисуется форма редактирования с помощью Javascript в правой части экрана с заполненнными полями; при клике на кнопку сохранения уходит AJAX-запрос на сервер на изменение заметки.

- при клике на крестик у заметки уходит AJAX запрос на сервер и заметка удаляется.

- при клике на кнопку создания новой заметки, в правой части экрана рисуется форма; после ввода данных и нажатия на кнопку сохранения уходит AJAX запрос на сервер, создается заметка и вновь созданная заметка добавляется в список в левой части экрана (списке)
